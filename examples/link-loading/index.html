Link & loading

Right-click the area below to open a menu with link items and an action that shows loading state.

Right-click or long-press here.

Links: Use type: "link" with href, optional target and rel. Ctrl/Cmd+click opens in a new tab.
Loading: Set loading: true on an item, then use updateMenu to clear it. Customize via config.spinner or per item: loadingIcon, loadingSize, loadingSpeed.
